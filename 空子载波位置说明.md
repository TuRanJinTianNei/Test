## 设计目的

**空子载波的设计目的（所有空子载波都不传输数据）：**

1. **DC子载波（索引1，空子载波）**: 避免直流分量，通常设为0
2. **低频保护带（索引2-28，空子载波）**: 防止频谱泄漏到负频率区域，作为频谱左侧保护带
3. **中间空子载波（索引229-285，空子载波）**: 数据子载波与共轭镜像之间的隔离，防止相互干扰
4. **高频保护带（索引486-512，空子载波）**: 防止频谱泄漏到高频区域，符合频谱掩码要求，作为频谱右侧保护带

## MATLAB代码验证

```matlab
IFFT_bin_length = 512;
carrier_count = 200;
carriers = (1:carrier_count) + (floor(IFFT_bin_length/4) - floor(carrier_count/2));
% carriers = [29:228]

conjugate_carriers = IFFT_bin_length - carriers + 2;
% conjugate_carriers = [286:485]

% 空子载波位置
null_carriers = setdiff(1:IFFT_bin_length, [carriers, conjugate_carriers]);
% null_carriers 应该包含 [1:28, 229:285, 486:512]
```
