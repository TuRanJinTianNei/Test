# OFDM系统仿真项目

## 项目概述

本项目是一个完整的OFDM（正交频分复用）通信系统仿真平台，实现了从发送端到接收端的完整链路，包括16QAM调制、IFFT/FFT变换、循环前缀、加窗处理、信道传输和误码率分析等功能。

## 系统架构

```
发送端 → 信道 → 接收端
  ↓       ↓       ↓
比特流   AWGN   比特流
16QAM    噪声   16QAM
IFFT            FFT
加CP            去CP
加窗            解调
```

## 核心系统参数（LTE标准10MHz系统）

### 一、基础固定参数（与子帧数无关）

| 参数名称 | 符号 | 数值 | 说明 |
|---------|------|------|------|
| **子载波间隔** | Δf | 15 kHz | LTE基本间隔，对应采样率30.72MHz |
| **系统带宽** | - | 10 MHz | LTE标准10MHz系统带宽 |
| **符号有效时长** | T | ≈66.67 μs | 1/15kHz，不含CP的有效符号时长 |
| **采样率** | fs | 30.72 MHz | 对应10MHz系统带宽 |
| **调制方式** | - | 16QAM | 每个子载波承载4比特 |
| **每子载波比特数** | bits_per_symbol | 4 | 16QAM调制阶数 |

### 二、子载波与频域参数（固定配置）

| 参数名称 | 符号 | 数值 | 说明 |
|---------|------|------|------|
| **IFFT点数** | N | 2048 | IFFT/FFT变换的点数（对应10MHz系统） |
| **总子载波数** | - | 600 | 索引范围-299~300（标准索引k） |
| **有效子载波数** | - | 504 | 索引范围-249~250（不含DC，传输数据） |
| **空子载波总数** | - | 96 | 含1个DC子载波 + 95个保护带 |
| **DC子载波** | - | 1 | k=0（MATLAB bin=1），设为0 |

**子载波索引映射：**
- 标准索引k: -299 ~ 300，其中k=0为DC
- MATLAB bin映射: k=0→bin=1, k>0→bin=k+1, k<0→bin=N+k+1
- 有效子载波：k = -249 ~ 250（共504个，不含DC）

### 三、帧结构及时域累计参数（100个子帧）

| 参数名称 | 符号 | 数值 | 说明 |
|---------|------|------|------|
| **子帧数** | - | 100 | 仿真中使用的子帧数量 |
| **单个子帧时长** | - | 1 ms | LTE标准子帧时长 |
| **100个子帧总时长** | - | 100 ms | 总仿真时长 |
| **每个子帧时隙数** | - | 2 | 每个子帧含2个时隙（0.5ms/时隙） |
| **100个子帧总时隙数** | - | 200 | 总时隙数 |
| **单个时隙OFDM符号数** | - | 7 | 常规CP：7个符号/时隙 |
| **100个子帧总OFDM符号数** | total_symbols | 1400 | 100 × 2 × 7 = 1400个符号 |
| **总传输比特数** | total_bits | ≈2.8224 Mbit | 1400符号 × 2016bit/符号 |

### 四、循环前缀（CP）参数（常规CP，100个子帧累计）

| 参数名称 | 符号 | 数值 | 说明 |
|---------|------|------|------|
| **长CP符号** | - | 160采样点 | 时隙第1个符号，时长5.208μs |
| **短CP符号** | - | 144采样点 | 时隙第2~7个符号，时长4.6875μs |
| **每个子帧长CP数** | - | 2 | 2时隙 × 1 = 2个 |
| **每个子帧短CP数** | - | 12 | 2时隙 × 6 = 12个 |
| **100个子帧长CP总数** | - | 200 | 累计长CP符号数 |
| **100个子帧短CP总数** | - | 1200 | 累计短CP符号数 |
| **总CP时长（100个子帧）** | - | ≈6.64 ms | 200×5.208μs + 1200×4.6875μs |

### 五、加窗设计参数（100个子帧）

| 参数名称 | 长CP符号（200个） | 短CP符号（1200个） | 说明 |
|---------|------------------|-------------------|------|
| **窗函数类型** | 升余弦窗 | 升余弦窗 | LTE标准升余弦窗 |
| **过渡长度（采样点）** | 40 | 36 | 长CP: 160/4, 短CP: 144/4 |
| **单符号总采样点数** | 2208 | 2192 | 长CP: 2048+160, 短CP: 2048+144 |
| **100个子帧总采样点数** | - | 2,872,800 | 200×2208 + 1200×2192 |

**加窗原理说明：**
- 目的：抑制符号边缘的陡峭跳变，减少带外辐射（频谱泄露）
- 重叠机制：符号拼接时存在可控的重叠，重叠区域限制在循环前缀（CP）内
- 符号结构：`[CP | 主体(N) | 后缀(过渡长度)]`
- 重叠区域：当前符号的后缀与下一个符号的CP前过渡长度个样本重叠相加
- 接收端：去除CP时只取主体部分，不受重叠影响

### 六、子载波分配（LTE标准）

#### 6.1 有效数据子载波
- **标准索引范围**: k = -249 ~ 250（共504个子载波，不含DC）
- **MATLAB bin范围**: 根据映射规则计算
- **类型**: 承载16QAM调制数据
- **每符号数据比特数**: 504 × 4 = 2016 bit/符号

#### 6.2 空子载波分布（共96个）

| 标准索引k | MATLAB bin | 类型 | 数量 | 说明 |
|----------|-----------|------|------|------|
| **0** | 1 | DC子载波 | 1 | 直流分量，设为0 |
| **-299 ~ -250** | - | 低频保护带 | 50 | 频谱左侧保护带，避免干扰 |
| **251 ~ 300** | - | 高频保护带 | 50 | 频谱右侧保护带，避免干扰 |

**验证**: 600 = 504 + 96 ✓（504有效 + 1 DC + 95保护带 = 600总子载波）

### 七、信道参数

| 参数名称 | 符号 | 数值 | 说明 |
|---------|------|------|------|
| **信道类型** | - | AWGN | 加性高斯白噪声信道（保持原高斯信道） |
| **目标SNR（详细分析）** | targetSNRdB | 15 dB | 用于详细分析的SNR点 |
| **SNR范围（BER曲线）** | SNR_range | 10-30 dB | BER-SNR性能曲线范围 |
| **SNR步进** | - | 2 dB | BER曲线计算步进 |

### 八、性能指标

| 指标名称 | 说明 |
|---------|------|
| **BER（误码率）** | 比特错误率，用于评估系统性能 |
| **MSE（均方误差）** | 发送信号与接收信号的均方误差 |
| **频谱效率** | 有效数据子载波数 × bits_per_symbol / (N + 平均CP长度) |
| **每符号有效数据比特数** | 2016 bit/符号 | 504个有效子载波 × 4bit/子载波 |
| **100个子帧总传输比特数** | ≈2.8224 Mbit | 1400符号 × 2016bit/符号（无编码） |

### 修改IFFT点数
```matlab

% 注意：需要相应调整carrier_count和carriers索引
```

### 修改调制方式


```matlab

targetSNRdB = 20;    % 修改详细分析的SNR点
```

## 理论背景

### OFDM原理
- **正交性**: 子载波间隔 Δf = 1/T，使得各子载波在彼此中心处为零
- **循环前缀**: 将符号尾部复制到开头，对抗多径干扰
- **加窗**: 平滑符号边缘，降低频谱旁瓣

### 16QAM调制
- **星座点数**: 16个星座点
- **每符号比特数**: 4比特
- **Gray编码**: 相邻星座点仅差1比特，降低误码率

### 瑞利分布（test6.m）
- **推导**: 二维高斯分布 → 瑞利分布（幅度）
- **应用**: 无线通信中的瑞利衰落信道

## 性能指标参考值

在SNR = 15 dB条件下（典型值）：
- **BER**: 约 10⁻⁴ ~ 10⁻³（取决于具体实现）
- **频谱效率**: 约 1.56 bit/s/Hz（200 × 4 / (512 + 128)）

## 注意事项

1. **随机种子**: 程序使用固定随机种子（`rand('twister',0)`）以保证可重复性
2. **内存占用**: 大样本数（1000符号）可能需要较大内存
3. **计算时间**: 完整仿真（包括BER曲线）可能需要几分钟
4. **MATLAB版本**: 建议使用MATLAB R2016b或更高版本
